Welcome to the License Code Generation API!

This API provides endpoints to generate and validate unique license codes.

API Endpoints

Generate Access Key

URL: https://lisencecodegen.onrender.com/generate_access_key
Method: GET
Success Response: Returns the generated access key. Code: 200, Content: { "access_key": "S3KLA7D9Z1" }

Generate License Code

URL: https://lisencecodegen.onrender.com/?access_key=ACCESS_KEY&action=generate&days=DAYS
Method: GET
URL Params:
- access_key=string: Your valid access key.
- action=generate
- days=integer: Number of days the license code should be valid for. If not provided, it will default to 30 days.
Success Response: Returns the generated license code. Code: 200, Content: { "license_code": "1F2E9R3T5Y" }

Validate License Code

URL: https://lisencecodegen.onrender.com/?access_key=ACCESS_KEY&action=validate&code=CODE
Method: GET
URL Params:
- access_key=string: Your valid access key.
- action=validate
- code=string: License code you want to validate.
Success Response: Returns a success message if the code is valid and hasn't been used. Code: 200, Content: { "message": "Code validated successfully" }

Note: Replace ACCESS_KEY, DAYS, and CODE with your actual access key, desired number of days, and license code respectively.

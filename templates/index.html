License Code Generator

Endpoints Documentation

  • To generate a new license code: /?access_key=<your_access_key>&action=generate&days=<days>

    This will generate a new license code which is valid for the provided number of days. If days is not provided, it defaults to 30 days.

  • To validate a license code: /?access_key=<your_access_key>&action=validate&code=<code_to_validate>

    This will validate the provided license code and return a message indicating whether the code is valid, already used, or expired.

  • To generate a new access key: /generate_access_key

    This will generate a new access key that can be used to generate and validate license codes.

License Code Generator

This application generates unique license codes and allows you to validate them.

Endpoints:

  • /generate_access_key

    Generates a new access key.

  • /license?access_key=ACCESS_KEY&action=generate&days=DAYS

    Generates a new license code. Replace ACCESS_KEY with your access key and DAYS with the number of days before the code expires.

  • /license?access_key=ACCESS_KEY&action=validate&code=CODE

    Validates a license code. Replace ACCESS_KEY with your access key and CODE with the license code you want to validate.

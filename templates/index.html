API Documentation

Generate an Access Key

URL: https://lisencecodegen.onrender.com/?action=generate_key

Method: GET

Success Response: 200 OK, {"access_key": "generated_access_key"}

Generate a License Code

URL: https://lisencecodegen.onrender.com/?access_key=YOUR_ACCESS_KEY&action=generate&days=NUMBER_OF_DAYS

Method: GET

Success Response: 200 OK, {"license_code": "generated_license_code"}

Validate a License Code

URL: https://lisencecodegen.onrender.com/?access_key=YOUR_ACCESS_KEY&action=validate&code=LICENSE_CODE

Method: GET

Success Response: 200 OK, {"message": "Code validated successfully"}
